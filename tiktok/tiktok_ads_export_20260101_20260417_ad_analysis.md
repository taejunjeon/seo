# TikTok Ads Export 분석 - 2026-01-01 ~ 2026-04-17 광고별

작성 시각: 2026-04-18 14:08 KST

## 결론

이번 파일은 **광고별 기간 합계 리포트**로는 쓸 수 있다. 광고 ID/광고명/광고그룹 ID가 들어 있어, TikTok 플랫폼 구매값이 어떤 광고 소재에 몰렸는지 확인하는 데 유용하다.

하지만 **일자별 리포트는 아니다.** `Date Created`는 광고 생성일이고, 우리가 필요한 `report_date`가 아니다. 따라서 이 파일만으로는 `tiktok_ads_daily` 적재, Guard 전후 일 단위 비교, 일자별 내부 장부 조인은 불가능하다.

## ROAS 정합성 검토

이 파일로 가능한 것:

1. TikTok 플랫폼 구매값이 어떤 광고 ID/광고명에 몰렸는지 확인
2. 추후 일자별 export와 결합했을 때 소재 단위 원인 분석 보조
3. 대표 보고에서 "어떤 광고가 플랫폼 구매값 대부분을 만들었다고 보고되는지" 설명

이 파일로 불가능한 것:

1. 일자별 ROAS 계산
2. Guard 적용 전후 비교
3. `tiktok_ads_daily` 적재
4. 운영 VM 원장과 동일 기간 확정매출 비교

특히 기간이 2026-01-01 ~ 2026-04-17이라 기존 운영 VM 원장 조회와 바로 맞지 않는다. 운영 VM `biocom_imweb` 원장은 같은 조건으로 조회해도 현재 2026-04-01 이후 데이터 2,516행만 반환된다. 그래서 이 파일의 1월~3월 플랫폼 구매값을 내부 장부와 공정하게 맞대려면 운영 VM 또는 별도 내부 장부의 같은 기간 데이터가 필요하다.

## 판정

이 파일은 **일자별 export의 대체재는 아니다.**  
다만 **광고 소재/광고 ID 단위 원인 분석 자료**로는 가치가 있다.

따라서 처리 우선순위는 다음과 같다.

1. 이번 파일은 `ad_range_total` 보조 자료로 보관
2. `/ads/tiktok`의 일자별 ROAS에는 사용하지 않음
3. 일자별 Custom report를 계속 요청
4. 일자별 파일 수령 후, 광고별 파일은 "어떤 광고가 gap을 만들었는지" 분석할 때 결합
