## 작업: 네비게이션 바 높이 최종 조정 (27인치 모니터 최적화)

### 변경 사항

#### 1. 바 높이
- 현재: h-[60px]
- 변경: h-[72px]

#### 2. 내부 요소 비례 조정 (바가 커진 만큼 함께 키워야 함)

로고 영역:
- 로고 아이콘: w-7 h-7 → w-8 h-8
- "Biocom AI Agent": text-[13px] → text-sm (14px)
- "AEO/GEO INTELLIGENCE": text-[9px] → text-[10px]
- 로고와 아이콘 사이 gap: gap-2.5 → gap-3

네비게이션 탭:
- 탭 텍스트: text-[13px] → text-sm (14px)
- 탭 패딩: px-4 py-2 → px-5 py-2.5
- 탭 간 간격: gap-1 → gap-1.5
- 활성 탭 pill의 border-radius는 rounded-full 유지

"Connected" 상태:
- 텍스트: text-xs → text-[13px]
- 초록 dot: h-2 w-2 → h-2.5 w-2.5

#### 3. 바 하단 그림자 강화
- 현재: shadow 0 1px 3px rgba(0,0,0,0.04)
- 변경: shadow 0 1px 4px rgba(0,0,0,0.06)
  (바가 커지면 아래 콘텐츠와의 분리감도 비례해서 강해져야 함)

### 주의사항
- 본문 콘텐츠 영역의 max-width, px 등은 변경하지 않음
- sticky 동작, backdrop-blur 등 기존 동작 유지
- 탭 클릭 로직 변경 없음


추가 관찰: 칼럼 분석 페이지에 빠져 있는 것
현재 칼럼 분석 페이지에 테이블만 있고 요약 KPI가 없습니다. 오버뷰에는 총 클릭수, 평균 CTR 등이 있는데, 칼럼 분석에 들어오면 바로 상세 테이블로 시작합니다. 사용자가 이 페이지에 와서 가장 먼저 알고 싶은 것은 "전체 칼럼 중 상위 성과는 뭐고, 얼마나 잘 되고 있나?"인데, 그 요약 없이 바로 수십 개의 행에 빠지게 됩니다.
테이블 위에 3~4개의 미니 KPI 카드를 넣는 것을 추천합니다. 예를 들어 "총 칼럼 수: 47개", "클릭 발생 칼럼: 12개", "TOP 10 평균 CTR: 2.8%", "종합 스코어 평균: 28점" 같은 핵심 요약입니다. 이것만으로도 페이지 진입 시 "아, 47개 칼럼 중 12개만 클릭이 발생하고, 평균 스코어가 28점이구나 — 아직 갈 길이 멀다"는 인사이트를 3초 만에 얻을 수 있습니다.
