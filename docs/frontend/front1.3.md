하단 "AI 에이전트 활동 상태" 섹션을 고급화해줘.

[현재 상태]
- 단순 불릿 리스트 (빨간점/주황점/파란점 + 텍스트)
- "실시간" 뱃지가 상단 스코어와 중복 디자인
- 타입별 구분(클릭, 콘텐츠, 브랜드, 기술)이 시각적으로 약함

[변경 방향]
1. 각 활동 항목을 개별 카드로 분리 (flex row, gap-3):
   - 좌측: 카테고리 아이콘 (클릭→MousePointerClick, 콘텐츠→FileText, 
     브랜드→Building2, 기술→Wrench) + 색상 원형 배경
   - 중앙: 제목 (bold) + 설명 텍스트 (muted)
   - 우측: 우선순위 뱃지 (높음/중간/낮음) 또는 timestamp

2. 우선순위별 좌측 border 색상:
   - 빨간점(긴급) → border-left: 3px solid #EF4444
   - 주황점(주의) → border-left: 3px solid #F59E0B
   - 파란점(참고) → border-left: 3px solid #3B82F6

3. 섹션 헤더 "AI 에이전트 활동 상태" 옆에:
   - "분석: 2026.2.13 오후 9:54:52" → relative time ("방금 전", "5분 전") 형태로
   - 새로고침 아이콘 버튼 추가

4. 카드 진입 시 stagger 애니메이션 (위에서 아래로 0.1s 간격 fade-in-up)

기존 데이터 구조와 라우팅은 유지해줘.
