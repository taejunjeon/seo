## 2) 저장소 반영/구현 현황(현재 상태)

### 2.1 백엔드(이미 “안정 버전” 접근을 사용 중)

- 엔드포인트: `GET /api/ga4/funnel`
- 구현: `runReport` 기반(퍼널 전용 API 미사용)
- 현재 퍼널 단계(범용)
  - 유기 검색 유입(Organic Search sessions)
  - 페이지 조회(screenPageViews)
  - 참여 세션(engagedSessions)
  - 전환 완료(conversions)
- 장점
  - `runFunnelReport`를 쓰지 않으므로 **요구사항의 “1안(추천)”과 구현 방향이 일치**
- 한계
  - 요구사항의 핵심인 **`content_to_product_click`**, **검사/영양제 분리**, **결제 시작→완료** 등 “의사결정용 단계”가 아직 퍼널에 들어가 있지는 않음

참고 파일:

### 2.2 프론트엔드(UI에 퍼널 “시각화 + 드롭오프”는 이미 존재)

- 위치: 사용자 행동 탭의 “SEO → 전환 퍼널”
- 현재 표시 내용
  - 단계별 바(%) + 절대값
  - 핵심 KPI(전체 전환율/페이지-세션/참여율/최대 이탈 구간)

- 한계

## 5) `content_to_product_click` 이벤트 추가 가이드(필수)

왜 필요한가:
- GA4 Data API만으로는 “칼럼에서 상품으로 이동” 같은 **순차 경로(이전 페이지 → 다음 페이지)**를 안정적으로 계산하기가 어렵습니다.
- 따라서 “콘텐츠 → 상업 행동”을 잇는 브릿지로 `content_to_product_click`이 사실상 필수입니다.

어디에 추가하나:
- `biocom.kr`(아임웹) 페이지에서 “칼럼/소개 페이지에서 상품·검사 페이지로 이동하는 링크 클릭”을 트래킹해야 합니다.

구현 방법(권장: GTM)
- Trigger
  - 페이지: `/healthinfo*`, `/what_biohacking*` 등 콘텐츠 영역
  - 조건: 상품/검사 페이지로 가는 링크 클릭(예: `/shop_view`, 특정 구매 URL 등)
- Tag: GA4 Event
  - `event_name`: `content_to_product_click`
  - 권장 파라미터(가능한 범위에서)
    - `content_path`: 클릭이 발생한 콘텐츠 경로
    - `target_path`: 이동 대상 경로
    - `target_type`: `product` | `test` | `etc`
    - `content_title`: 가능하면(없으면 생략)

검증(최소 완료 기준)
1) GA4 DebugView에서 이벤트 발생 확인
2) GA4 Realtime/Events 보고서에서 이벤트 집계 확인
3) GA4 Data API에서 `sessions(eventName="content_to_product_click") > 0` 확인

## 11) 아직 결정 안 된 질문 3개(Q1~Q3)

UI 템플릿 [6]에 들어갈 “미결정 질문” 초안입니다. (결정되면 퍼널 설계/구현이 확정됩니다)

### Q1. “구매 완료”를 어떤 이벤트로 확정할까요?

현재 GA4에 `purchase`, `purchase_complete`, `결제완료` 등이 함께 존재합니다.  
퍼널/전환율을 한 가지로 고정하려면 최종 완료 이벤트를 **1개로 통일**해야 합니다. (권장: GA4 표준 `purchase`)

### Q2. `content_to_product_click`의 “대상 URL/타입” 범위를 어디까지로 할까요?

예: `/shop_view?idx=...`는 상품, 검사 상품도 동일 패턴인지, 다른 결제/랜딩 URL이 존재하는지.  
또한 `target_type`을 `product/test/etc`로 나눌 때 어떤 규칙으로 분류할지 결정이 필요합니다.

### Q3. 퍼널 지표는 “sessions / users / eventCount” 중 무엇으로 고정할까요?

현재 문서/로직은 **세션(sessions) 기반**을 전제로 했습니다(해석 안정성).  
다만 조직 KPI가 “사용자(Users)”나 “이벤트(클릭 수)” 중심이면 퍼널 정의가 달라집니다. “의사결정용” 1안 기준을 확정해야 합니다.
