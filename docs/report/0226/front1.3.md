## 대상

## 변경 1: "자세히 보기" 링크 위치 개선

현재 "자세히 보기" 링크가 눈에 잘 안 띈다.
카드 헤더 우측("최근 30일" 옆)에 "→ 상세 분석" 링크를 배치해라.
기존 하단 링크가 있다면 제거하고 헤더로 이동.

## 변경 2: 전주 대비 변화량 표시

현재 KPI(세션 215, 활성 사용자 25 등)에 변화량이 없다.

구현 방법:
- API 호출을 2번 해라:
  1. 최근 30일 (현재)
  2. 이전 30일 (비교)
- 각 KPI 옆에 변화량 표시: "+12% ▲" (초록) 또는 "-5% ▼" (빨강)
- 이전 기간 데이터가 없으면 변화량 표시 생략

단, API 호출이 2배가 되므로 성능이 우려되면
이전 기간 호출은 생략하고 TODO로 남겨라.
