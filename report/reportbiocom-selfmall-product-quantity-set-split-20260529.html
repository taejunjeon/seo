권장 분배 방식

추천 v0

수량은 구성품으로 분배, 매출은 원상품에도 남긴다

첫 버전에서는 매출 보고 숫자를 흔들지 않기 위해 “세트 매출”은 기존 상품에도 유지하고, 운영·재고·인기상품 판단용으로만 구성품 수량을 추가 계산하는 방식이 안전합니다.

예: `검사 SET 1` 94개는 매출은 SET 상품에 남기고, 구성품 수량에는 대사기능 94건과 음식물 과민증 94건을 추가합니다.

v1 이후

매출도 구성품 가격 비율로 나눈다

구성품별 매출까지 필요하면 정가 또는 관리자가 정한 비율로 나눌 수 있습니다. 다만 이 단계는 할인·공구·구독 조건 때문에 기준이 흔들릴 수 있어, 먼저 수량 분배를 고정한 뒤 진행하는 편이 좋습니다.

정가 비율, 균등 분배, 수동 비율 중 하나를 상품군별로 선택해야 합니다.

수동 매핑표 설계

사람이 정해줘야 하는 최소 정보

1. 어떤 상품/옵션을 세트로 볼지

상품번호, 상품명, 옵션명 패턴을 정합니다. 예: 상품번호 328 + `1인 검사 SET`.

2. 세트 1개가 어떤 구성품 몇 개로 풀리는지

예: 대사기능 1개, 음식물 과민증 1개. 3+1 상품은 같은 제품 4개.

3. 매출을 나눌지, 수량만 나눌지

추천은 수량 먼저입니다. 매출 분배는 다음 단계에서 가격 비율을 확정합니다.

필드 의미
