# 더클린커피 Meta 광고 불일치 분석

> **기준일**: 2026-04-05
> **문제**: 더클린커피가 월 ₩606만을 Meta 광고에 집행 중인데, 우리 솔루션에서는 "광고 미집행"으로 표시됨

---

## 한 줄 결론

**더클린커피 실제 광고 계정과 우리 API에 연결된 계정이 다르다.** 실제 광고가 돌아가는 계정(`act_1461947384242612`)에 우리 Meta 토큰의 접근 권한이 없어서, 우리 대시보드에서 더클린커피 광고 데이터가 안 보이는 것이다.

---

| 노출 | 185,758 |
| 클릭 | 4,739 |

| 전환 매출 | **₩9,750,770** |
| ROAS | **1.61x** |
| 평균 객단가 | ₩39,003 |
| 클릭율 (CTR) | 2.55% |
| 구매전환율 | 5.28% |

캠페인 구성:

### 스크린샷에서 확인된 계정

- Meta 광고 관리자 스크린샷의 계정 ID: **`1461947384242612`**
- "새 매출 캠페인" 1개가 **임시 저장** 상태. 결과/노출/지출 모두 "—"
- **이 계정에서도 현재 광고가 돌아가고 있지 않음**
- 즉, 3월에 ₩606만을 집행한 계정은 이 계정이 아니거나, 3월에 돌리고 4월에 중단한 것

### 우리 솔루션에 등록된 계정

| 계정 ID | 이름 | 상태 | 광고 실적 |

### 실제 광고 계정 직접 조회 시도

```
GET /api/meta/insights?account_id=act_1461947384242612
→ 오류: "Ad account owner has NOT grant ads_management or ads_read permission"
```

**우리 토큰에 해당 계정 접근 권한이 없음.**

## 2. 원인 분석

```
더클린커피 Meta 광고 계정이 최소 3개 존재:

1. act_1382574315626662 (우리 API에 연결됨, ACTIVE, 데이터 없음)
   → 아마 오래전에 만들어졌거나, 다른 용도의 빈 계정

2. act_853441229535835 (우리 API에 연결됨, CLOSED)
   → 이전에 사용하다가 종료됨

3. act_1461947384242612 (스크린샷, 실제 광고 집행 중)
   → 현재 마케팅팀이 실제로 사용하는 계정
   → 우리 토큰에 권한이 없음 → 데이터 조회 불가
```

**가능성 3가지:**

1. **3월 광고는 `act_1461947384242612`에서 했지만, 4월에 중단함** — 스크린샷에서 "새 매출 캠페인"만 임시저장 상태. 3월 캠페인은 종료/삭제되었을 수 있음.
2. **3월 광고는 또 다른 계정에서 했음** — 더클린커피 관련 계정이 3개 이상 존재할 가능성. 마케팅 담당자에게 확인 필요.
3. **외부 대행사 계정에서 집행** — xlsx 보고서를 만든 마케터가 자신의 Meta 계정이나 대행사 계정에서 더클린커피 광고를 돌렸을 수 있음. 이 경우 우리 Business Manager와 완전히 별개.

바이오컴도 같은 패턴이었음: 처음에 `act_3744078992317776`("정희용")으로 연결했다가, 실제 계정 `act_3138805896402376`("바이오컴_임시")으로 수정.

## 3. 해결 방법

### 즉시 확인 필요 (TJ님 액션)

**방법 A: Meta Business Suite에서 권한 추가**

1. https://business.facebook.com/settings 접속
2. 좌측 메뉴 "계정" > "광고 계정" 클릭
3. `1461947384242612` 계정 선택
4. "사람" 또는 "파트너" 탭에서 우리 앱/토큰에 `ads_read` 권한 부여
5. 또는 해당 계정에 연결된 Business Manager에 우리 System User를 추가

**방법 B: 해당 계정의 토큰으로 교체**

1. Meta 광고 관리자에서 `1461947384242612` 계정에 접근할 수 있는 사용자로 로그인
2. Graph API Explorer에서 해당 계정의 access token 발급
3. `.env`의 `META_ADMANAGER_API_KEY`를 이 토큰으로 교체 (단, 바이오컴/AIBIO도 같은 토큰으로 접근 가능해야 함)

**방법 C: 하나의 Business Manager로 통합**

바이오컴, 더클린커피, AIBIO의 광고 계정이 모두 같은 Business Manager 아래에 있으면 하나의 토큰으로 전부 조회 가능. 현재는 더클린커피 계정이 다른 Business Manager에 속해 있을 가능성.

## 5. 3월 더클린커피 Meta 성과 분석 (xlsx 기준)

권한 문제가 해결되면 API로 직접 가져올 수 있지만, 현재는 xlsx 데이터로 분석.

### 성과 요약

| 지표 | 더클린커피 | 팀 키토 (비교) |
|------|----------|------------|
| 광고비 | ₩606만 | ₩304만 |
| 전환 매출 | ₩975만 | ₩707만 |
| **ROAS** | **1.61x** | **2.33x** |
| 구매 수 | 250건 | 92건 |
| 평균 객단가 | ₩39,003 | ₩76,879 |
| CTR | 2.55% | 2.57% |
| 구매전환율 | 5.28% | 2.24% |
| CPC | ₩1,279 | ₩742 |

### 인사이트

1. **ROAS 1.61x는 낮음**: 마진율 40% 기준 손익분기 ROAS가 2.5x. 현재 첫 구매에서 적자.
2. **그러나 구매전환율 5.28%는 매우 높음**: 팀 키토(2.24%)의 2.4배. 클릭한 사람이 잘 산다.
3. **문제는 CPC ₩1,279**: 팀 키토(₩742)의 1.7배. 클릭 비용이 비쌈.
4. **평균 객단가 ₩39,003**: 커피라서 낮음. CPC가 비싸면 ROAS가 떨어질 수밖에 없음.
5. **재구매율이 관건**: 커피는 소모품이라 재구매율이 검사키트보다 높을 수 있음. 아임웹 기준 9.4%(3개월), PlayAuto 기준 21.7%(12개월). 재구매 반영 시 LTV/CPA가 개선됨.

### 4월 예산 계획 (xlsx)

| 채널 | 예산 | 목표 ROAS |
|------|------|---------|
| Meta | ₩100만 | 3.0x |
| 네이버 | ₩250만 | 5.0x |
| 쿠팡 | ₩150만 | 4.0x |
| **합계** | **₩500만** | |

**3월 대비 Meta 예산을 ₩606만 → ₩100만으로 대폭 축소.** ROAS 1.61x가 낮았기 때문으로 보임.

## 6. 우리 솔루션에서 더클린커피 Meta 데이터를 보려면

```
현재:
  우리 토큰 → act_1382574315626662 (빈 계정) → 데이터 없음

해결 후:
  우리 토큰 → act_1461947384242612 (실제 계정) → 3월 실적 표시
```

코드 변경은 1줄:

```typescript
// frontend/src/app/ads/page.tsx 및 backend/src/routes/meta.ts, ads.ts
{ site: "thecleancoffee", account_id: "act_1461947384242612" }
```

**단, 토큰 권한 추가가 선행되어야 함.** 권한 없이 계정 ID만 바꾸면 API 오류가 남.
