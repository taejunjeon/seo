## 2. 이번 구현 범위 (Phase 1)

### 2-1. 카카오톡 스타일 실시간 미리보기
- Step 3 (미리보기) 영역에 폰 프레임 + 카카오톡 말풍선 UI 추가
- 메시지 입력 시 실시간 반영
- 템플릿 변수 치환 결과 표시
- 버튼 미리보기 (카카오 노란 버튼 스타일)
- SMS일 때는 SMS 스타일 말풍선

### 2-2. 메시지 종류 배지
- 알림톡 채널 선택 시 광고성(AD)/정보성(BA) 구분 표시
- 선택된 템플릿의 타입에 따라 자동 반영

### 2-3. 글자수 카운터 (알림톡)
- 알림톡 메시지 본문 글자수 표시 (1,000자 제한)
- 변수 치환 전/후 글자수 표시

### 2-4. 링크 버튼 편집
- 템플릿 선택 후 버튼 URL 편집 가능
- 링크 버튼 추가/삭제 (최대 5개, Aligo 제한)

### 2-5. 발송 비용 표시
- 알림톡: 건당 ~₩8 표시
- SMS/LMS: 기존 바이트 기반 비용에 잔여 포인트 대비 표시
