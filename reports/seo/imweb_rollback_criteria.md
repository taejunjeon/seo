# 아임웹 URL 정리 작업 — 롤백 기준 (정본)

**대상 작업**: B 승인안 — 아임웹 URL 정리 (canonical / noindex / robots.txt)
**적용 단위**: 단계별 롤백 가능 (전체 롤백 불필요. 문제가 생긴 단계만 되돌림)
**판단 주기**: 적용 후 24시간 / 7일 / 14일 시점

---

## 1. 즉시 롤백 신호 (적용 후 0~24시간 내)

| 신호 | 측정 위치 | 임계값 | 대응 |
|---|---|---|---|
| 사이트 전체 트래픽 급락 | GA4 실시간 + 일일 비교 | 전일 대비 -30% 이상 | robots.txt 백업본 즉시 교체 |
| robots.txt 자체 응답 오류 | `curl -I https://biocom.kr/robots.txt` | HTTP 200 아님 | 백업본 교체 |
| sitemap.xml 응답 오류 | `curl -I https://biocom.kr/sitemap.xml` | HTTP 200 아님 또는 0 byte | sitemap 설정 원복 |
| 핵심 6개 페이지 「색인 안 됨」 분류 | GSC URL 검사 | 1개라도 그렇게 분류되면 | 해당 페이지 noindex 즉시 해제 |
| 사용자 항의·CS 문의 폭증 | 고객센터 / Slack | 평소 대비 5건 이상 증가 | 변경 범위 점검 → 부분 롤백 |

---

## 2. 1주일 후 롤백 신호 (적용 후 7일)

| 신호 | 측정 위치 | 임계값 | 대응 |
|---|---|---|---|
| 「색인됨」 페이지 수 급감 | GSC > 색인 > 페이지 | 적용 전 대비 -30% 이상 | noindex 적용 범위 재확인. 의도하지 않은 페이지가 차단됐는지 1줄씩 점검 |
| 클릭(GSC 7일) 급감 | GSC 라이브 KPI | 적용 전 7일 대비 -25% 이상 | 어느 페이지에서 클릭이 빠졌는지 page 차원 분석 → 해당 페이지 처리 원복 |
| 평균 순위 급락 | GSC 라이브 KPI | 적용 전 7일 대비 +2계단 이상 (낮을수록 좋음) | canonical 통일이 잘못된 URL을 가리키지 않았는지 점검 |
| 매출·전환 하락 | 아임웹 통계 / GA4 | 7일 평균 -20% 이상 | 상품 페이지 noindex/canonical 점검 (가장 흔한 원인) |

---

## 3. 2주 후 검증 (적용 후 14일)

이 시점에서 정상 신호가 나오면 작업 성공:

| 신호 | 측정 위치 | 정상 기준 |
|---|---|---|
| 같은 키워드로 1개 URL만 노출 | "바이오밸런스" 등 상품명 검색 | 같은 상품의 다른 URL이 동시에 안 뜸 |
| GSC URL 검사 표준 URL 일치 | GSC > URL 검사 | 사용자가 선언한 표준 URL = Google이 선택한 표준 URL |
| 잡음 URL이 검색결과에서 사라짐 | site:biocom.kr ?q= 등 검색 | 결과 0건 또는 급감 |
| 클릭/CTR 안정 또는 개선 | GSC 라이브 KPI | 적용 전 대비 ±10% 내 안정 (개선 신호는 4~8주 후) |

---

## 4. 부분 롤백 매핑 (어느 단계가 문제면 어디만 되돌릴지)

| 문제 신호 | 의심 단계 | 부분 롤백 |
|---|---|---|
| 핵심 상품 페이지 「색인 안 됨」 | §2-1 noindex 적용 | 해당 페이지 noindex 체크만 해제 |
| 같은 상품 검색에서 우리 페이지 0건 | §2-2 canonical 통일 | 해당 상품 canonical을 자기 자신으로 되돌림 |
| robots.txt 응답 오류 | §2-3 robots.txt 수정 | 백업본 교체 |
| sitemap loc 수 급감 | §2-4 sitemap 정리 | sitemap 생성 설정 원복 |

---
