Meta Lead Form의 장점:
- 홈페이지 방문 불필요 → 전환율 높음 (앱 내에서 폼 완성)

**Meta Lead Form → 전화 영업 프로세스:**

```
Meta 영상 광고
  → 인스턴트 폼 제출 (이름, 연락처, 관심사)
  → Meta Ads Manager에서 리드 다운로드 (매일)
  → 전화 영업 리스트로 전달
  → 24시간 내 전화 (콜드 리드 → 체험 예약)
  → 방문 체험
  → 프로그램 등록
```

성과 측정:
- Meta Ads Manager에서 리드 수, CPL, 리드→체험 전환율 추적
- 전화 영업 결과는 수동 스프레드시트 또는 CRM 기록

## 아임웹 폼 추적이 우리 코드로 안 되는 이유 (기록)

V2~V5까지 시도한 결과:
1. **V2 (XHR/fetch 가로채기)**: 아임웹이 `XMLHttpRequest.prototype.open` 참조를 우리 스크립트보다 먼저 캐시하여 패치 무력화
2. **V3 (버튼 클릭 + sendBeacon)**: 아임웹 폼 버튼이 우리 click 리스너 패턴과 불일치 + 수백 건 중복 발생
3. **V4 (XHR form_add.cm 감지)**: V2와 동일 원인으로 실패
4. **V5 (PerformanceObserver)**: 콜백 타이밍이 맞지 않음

**해결**: 푸터에 `dataLayer.push({event:'aibio_form_submit'})` → GTM 맞춤 이벤트 트리거 → GA4 `generate_lead` 이벤트. 아임웹 `form_add.cm` 성공 시 PerformanceObserver가 dataLayer에 push하고, GTM이 이를 감지한다.
